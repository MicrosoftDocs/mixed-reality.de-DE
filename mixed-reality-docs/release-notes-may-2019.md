# <a name="release-notes---may-2019"></a>Anmerkungen zu dieser Version – Mai 2019

Die **[Windows 10 Oktober 2018 Update](https://blogs.windows.com/windowsexperience/2018/10/02/find-out-whats-new-in-windows-and-office-in-october/)** (auch bekannt als RS5) bietet neue Features für HoloLens und Windows Mixed Reality immersive Headsets mit PCs. 

Um auf das neueste Release für HoloLens oder PC (für Windows Mixed Reality immersive (VR)-Headsets) zu aktualisieren, öffnen die **Einstellungen** wechseln Sie zur app **Update und Sicherheit**, und wählen Sie dann die **überprüfen Updates** Schaltfläche. Auf einem Windows 10-PC, können Sie auch manuell installieren der Windows 10 Oktober 2018 Aktualisieren mit der [Tool zur Erstellung von Windows Media](https://www.microsoft.com/software-download/windows10).

**Neueste Release für den Desktop:** Windows 10 Oktober 2018 Update (**10.0.17763.107**)<br>
**Neueste Release für HoloLens:** Windows 10 Oktober 2018 Update (**10.0.17763.134**)<br>

## <a name="new-features-for-windows-mixed-reality-immersive-headsets"></a>Neue Features für immersive Headsets Windows Mixed Reality

Windows 10 Oktober 2018 Update enthält zahlreiche Verbesserungen für die Verwendung von Windows Mixed Reality immersive (VR) Headsets mit Ihrem desktop-PC.

### <a name="for-everyone"></a>Für alle Benutzer

* **Mixed Reality Taschenlampe** : Öffnen Sie ein Portal, in der realen Welt zum Suchen der Tastatur finden Sie unter einer Person in der Nähe, oder sehen Sie sich Ihre Umgebung ohne Ihre Kopfhörer! Sie können aktivieren Mixed Reality Taschenlampe über das Startmenü, durch Drücken von Windows + Ziehpunkte auf dem Controller während der Übertragung oder indem Sie sagen "Taschenlampe on/off". Zeigen Sie Ihren Controller in Richtung wie die Verwendung einer Taschenlampe im Dunkeln erhalten möchten.

    ![Mixed Reality Taschenlampe](images/mr-flashlight.png)

* **Neue apps und Möglichkeiten, um Inhalt in die gemischte Realität Startseite starten**
    * Bei Verwendung von [Windows Mixed Reality für SteamVR](https://docs.microsoft.com/windows/mixed-reality/enthusiast-guide/using-steamvr-with-windows-mixed-reality), Ihre Titel SteamVR jetzt angezeigt im Startmenü und die app-startfeldern von für die einzelnen in der mixed Reality home eingefügt werden können.
    
        ![App-startfeldern von SteamVR](images/steamvr-launchers.png)
        
    * Neue *360 Videos* app für die Ermittlung einer regelmäßig kuratierten Liste verfügbarer 360-Grad-Videos.
    * Neue *WebVR Showcase* -app für die Ermittlung einer regelmäßig kuratierten Liste verfügbarer WebVR kommt.
    * Erstmalige Windows Mixed Reality Kunden geben Sie die Cliff House und ermitteln, dass es mit 3D app-startfeldern für einige unserer bevorzugten immersive apps und Spiele, aus dem Microsoft Store vorab aufgefüllt.
    * Microsoft Edge-Windows enthalten nun einen *Freigabe* Schaltfläche.
* **Menü für schnelle Aktionen** -in eine immersive mixed Reality-app, Sie können drücken Sie in ein neues Menü für schnelle Aktionen mit einfachem Zugriff auf den Zugriff auf die Windows-Schaltfläche *SteamVR Menü*, *Foto/Video-Erfassung*, *Taschenlampe*, und *home*.
* **Unterstützung für Rucksack PCs** -Windows Mixed Reality immersive Headsets von (VR) auf Rucksack PCs ausführen, ohne einen Display-Emulator verwenden, nachdem Setup abgeschlossen wurde.
* **Neue Audiofunktionen** – Sie können jetzt die Audiodatei aus einer Windows Mixed Reality-Erfahrung sowohl die Audio-Jack (oder Kopfhörer) in Ihrer Kopfhörer spiegeln *und* ein audio-Gerät mit Ihrem PC (z. B. externe Lautsprecher) verbunden. Wir haben außerdem einen visuellen Indikator für die Volume-Ebene in Ihrer Kopfhörer des Anzeige hinzugefügt.
* **Weitere Verbesserungen**
    * Mixed Reality-Portal-Updates werden jetzt über den Microsoft Store übermittelt rascheren Aktualisierungen zwischen Hauptversionen von Windows aktivieren. Beachten Sie, dass dies gilt nur für die desktop-app und die Windows Mixed Reality Kopfhörer Erfahrung immer noch mit dem Betriebssystem aktualisiert. 
    * Wenn Headsets, die in den Ruhezustand versetzt werden, werden Windows Mixed Reality-apps anstelle von angehalten, beendet (bis Mixed Reality-Portal, geschlossen ist).

(Vergessen Sie nicht, Sie immer mit Sprachbefehlen zu möglich.)

## <a name="known-issues"></a>Bekannte Probleme

Wir haben hart daran gearbeitet, um eine hervorragende Windows Mixed Reality-Erfahrung zu bieten, aber verfolgen wir noch einige bekannte Probleme. Wenn Sie andere Benutzer finden, wenden Sie [senden Sie uns Feedback](https://docs.microsoft.com/windows/mixed-reality/give-us-feedback).

### <a name="hololens"></a>HoloLens
 
#### <a name="after-update"></a>Nach dem update
Sie können die folgenden Probleme feststellen, wenn mithilfe der Windows 10 Oktober 2018 aktualisieren, auf Ihre HoloLens:
* **Apps können in Anmeldung Schleife beim Starten von einer Benachrichtigung schließlich** – einige apps, die eine Anmeldung können End oben in einer Endlosschleife-Anmeldung ausgeführt, wenn von einer Benachrichtigung gestartet. Beispielsweise kann dies nach der Installation von Microsoft-Unternehmensportal-app aus dem Microsoft Store ein, und starten Sie ihn aus der Benachrichtigung zum Abschluss der Installation.
* **Seite "App-Anmeldung" kann mit einer leeren Seite abschließen** – In einigen Fällen, wenn eine anmeldeaufforderung über Ihre Anwendung beim Abschluss anzeigt, die Anmeldeseite wird nicht geschlossen, und stattdessen wird eine leere (schwarze) Seite. Sie können schließen Sie die leere Seite oder verschieben Sie es um die Anwendung unter zu gewinnen. Beispielsweise kann dies in Anmeldung während der MDM-Registrierung aus der Einstellungen-app auftreten.
